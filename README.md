# DU IA Finance & Actuariat - CARMA

Site statique du Diplôme Universitaire IA appliquée à la Finance et à l'Actuariat, adapté pour CARMA, Centre Africain de Recherche en Risk Management et Assurances.

## Pages disponibles

- `index.html` : syllabus détaillé de la formation.
- `cal.html` : calendrier pédagogique complet de la promotion 2026-2027.

Les deux pages sont autonomes : le CSS, le JavaScript et les données pédagogiques sont intégrés directement dans chaque fichier HTML. Aucune dépendance externe n'est requise.

## Documentation

- `docs/syllabus.md` : structure, contenu et maintenance de la page syllabus.
- `docs/calendrier.md` : structure, données et maintenance de la page calendrier.
- `docs/deploiement.md` : publication locale et GitHub Pages.

## Utilisation locale

Ouvrir directement `index.html` ou `cal.html` dans un navigateur moderne.

Pour servir le site en local :

```bash
python -m http.server 8000
```

Puis ouvrir :

- `http://localhost:8000/index.html`
- `http://localhost:8000/cal.html`

## Contenu intégré

Le site reprend les informations pédagogiques du DU IA Finance & Actuariat :

- 7 unités d'enseignement.
- Syllabus détaillé avec objectifs, cas pratiques et technologies.
- Calendrier d'octobre 2026 à juin 2027.
- 163 événements de cours, suivis, soutenances et rendus.
- 8 éléments spéciaux : jours fériés et vacances.

## Identité institutionnelle

Toutes les mentions institutionnelles ont été adaptées à CARMA. Le site ne dépend d'aucun logo, police ou ressource hébergée par un tiers.
